What is DAQ Expert?

The main goal is to increase data taking efficiency.

The main tasks are to perform automatic recovery and to automate routine actions in CMS currently done by the operator.

First version is suggesting the actions to shifters.

How to use DAQ Expert?

DAQ Expert provides Notification tool. It delivers analysis results in the form what's the problem + what's the best action to take designed for 2 use cases:

  • to be used in control room - as a single page view popping notifications in real time
  • to be used by DAQ Experts on the daily basis - as a notifications delivered via SMS/e-mails based on personal preferences

How DAQ Expert works?

DAQ Expert analyses the snapshots of DAQ system using expert knowledge. The results are visualized in browser tool.

  • StableBeams - identified beam mode
  • NoRate - identified no rate in DAQ system
  • Downtime - indicated downtime in CMS DAQ
  • FC5 identified known failure case - FED backpressured
